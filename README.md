# VRChat Avatar Explorer

VRChat Avatar Explorerは、VRChatのアバターを改変する際にファイルを見やすく整理するための便利なツールです。自分用に作成しましたが、もし使いたい方がいれば自由にお使いください。

## 特徴

- VRChatアバターのファイルを簡単に整理
- 多機能
